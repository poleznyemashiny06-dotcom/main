# Предпросмотр сайта (без знаний кода)

Сайт уже готов, его можно открыть **в 1 команду**.

## Самый простой запуск

```bash
./start_preview.sh
```

После запуска:
- браузер откроется автоматически;
- если не открылся — скопируйте ссылку из терминала (например, `http://127.0.0.1:8000/index.html`) и вставьте в браузер.

## Если нужна ручная команда

```bash
python3 preview.py
```

Дополнительно:
- другой порт: `python3 preview.py --port 9000`
- без авто-открытия браузера: `python3 preview.py --no-browser`
